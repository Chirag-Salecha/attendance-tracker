Theme: Algorithm Foundations & Complexity Analysis

Kinds of Problems in Nature

Iteration: Used for repeated operations like scanning arrays and attendance data.

Recursion: Breaks problems into subproblems, used in merge sort and tree traversal.

Backtracking: Explores all possibilities, used in puzzles and constraint problems.

Time and Space Complexity

Time Complexity measures growth of execution time with input size.
Space Complexity measures extra memory used.

Orders of Growth:
  • O(1)
  • O(log n)
  • O(n)
  • O(n log n)
  • O(n²)
  • O(2ⁿ)

Sorting Algorithms

  • Bubble Sort
  • Selection Sort
  • Insertion Sort
  • Merge Sort
  • Quick Sort
  • Heap Sort
