Business Case

Project: Student Attendance Pattern Tracker

Course: Design and Analysis of Algorithms (DAA)

Institution: KLE Technological University

1. Introduction

Educational institutions generate large volumes of student attendance data on a daily basis. Analyzing this data manually is time-consuming and prone to errors, especially in large classrooms. There is a clear need for an automated, algorithm-driven solution to analyze attendance patterns efficiently.

2. Business Problem

Faculty members need timely insights into student attendance behavior, such as:

  • Students with consecutive absences
  • Students falling below minimum attendance requirements
  • Irregular attendance trends over time

Without algorithmic analysis, these patterns may go unnoticed until academic performance is negatively affected.

3. Proposed Solution

The Student Attendance Pattern Tracker applies Design and Analysis of Algorithms principles to efficiently process attendance records, detect meaningful patterns, and generate academic insights that support early intervention and informed decision-making.

4. Individual Business Scenarios & Algorithmic Mapping

Scenario 1 – Consecutive Absence Detection (Chirag Salecha)

  • Problem: Identify students absent for multiple consecutive days
  • Data Structure: Arrays
  • Algorithm: Linear traversal
  • Time Complexity: O(n)

Scenario 2 – Low Attendance Percentage Analysis (Achyuth RH)

  • Problem: Identify students below the required attendance threshold
  • Data Structure: Arrays, Hash Maps
  • Algorithm: Frequency counting
  • Time Complexity: O(n)

Scenario 3 – Attendance Trend Monitoring (Shridevi VD)

  • Problem: Analyze consistency of attendance over time
  • Data Structure: Arrays
  • Algorithm: Sliding window technique
  • Time Complexity: O(n)

Scenario 4 – Attendance Ranking & Classification (Prajwal VB)

  • Problem: Rank students based on attendance regularity
  • Data Structure: Arrays
  • Algorithm: Sorting algorithms
  • Time Complexity: O(n log n)

8. Sample Code Example

The following C++ code demonstrates a simple algorithm to detect consecutive absences using linear traversal. This algorithm runs in O(n) time complexity. 

#include <iostream>
using namespace std;

int main() {
    int n = 7;
    int attendance[] = {1, 0, 0, 0, 1, 0, 1}; 
    // 1 = Present, 0 = Absent

    int consecutiveAbsences = 0;

    for(int i = 0; i < n; i++) {
        if(attendance[i] == 0) {
            consecutiveAbsences++;
            if(consecutiveAbsences >= 3) {
                cout << "Alert: Student has consecutive absences!" << endl;
                break;
            }
        } else {
            consecutiveAbsences = 0;
        }
    }

    return 0;
}

This example demonstrates how basic algorithmic techniques such as iteration and conditional logic can be applied to solve real-world academic problems efficiently.
